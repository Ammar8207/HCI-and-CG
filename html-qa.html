How to Use This Page

Read each question aloud and verify that your answers use correct semantic elements and accessibility practices. These items mirror common exam prompts.

Common Questions and Answers

Q1: Why use semantic elements like header, nav, main, and footer?
A1: They describe the structure of a page to browsers, screen readers, and search engines, improving accessibility, navigation, and maintainability.
Q2: What is the purpose of aria-label on a nav element?
A2: It provides an accessible name so screen readers can distinguish multiple navigation regions (e.g., “Primary navigation”).
Q3: When should aria-current="page" be used?
A3: Use it on the link that represents the current page so users of assistive technology know where they are in the site.
Q4: What is the difference between alt text and a caption?
A4: Alt text describes an image for users who cannot see it; a caption provides visible context or explanation for all users.
Q5: Why are labels required for form inputs?
A5: Labels ensure form controls are announced correctly by screen readers and make the clickable area larger for better usability.
Q6: What is the correct way to associate a label with an input?
A6: Use a label element with a for attribute that matches the input’s id.
Q7: What do thead and tbody provide in a table?
A7: They group header and body rows to improve structure, readability, and accessibility for assistive technologies.
Q8: Why use th scope="col" and th scope="row"?
A8: Scope tells screen readers which headers apply to each cell, making complex tables easier to understand.
Q9: What is the purpose of the main element?
A9: It identifies the unique, central content of a page and helps users skip repeated navigation.
Q10: When should you avoid using div and span?
A10: Avoid them when a semantic element exists, because semantic elements provide meaning and improve accessibility.

Quick Checklist

  • One h1 per page with logical heading order.
  • Navigation uses nav > ul > li > a.
  • Form inputs always have associated label.
  • Tables include caption, thead, tbody, and scoped headers.
  • Images include meaningful alt text.
